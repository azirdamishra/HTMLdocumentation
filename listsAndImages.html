Different types of lists in HTML :
NUMBERED/ ORDERED LIST: It is numbered in a proper sequence : --> i, a, A
UNNUMBERED LIST: It is also known as unordered list and starts with bullets
DEFINITION LIST: this is a list which is without numbers or bullets and there are specific terms that help to make the list

DEFINITION LIST

A definition list is not a list of single items.
It is a list of items with a description of each item
it is useful in works like a dictionary where there are words and their meanings listed without numbers or bullets

A Definition List starts with a DL tag
each term (Definition Term) starts with a DT tag
each description starts with a DD tag
