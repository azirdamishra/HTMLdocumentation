Both images and gifs can be used as background,

if the image is smaller than the page then the image will repeat itself.

the image tag is an empty tag. it has other attributes like src, alt, width, height, etc


to display an image on a page, we use src which stands for fource. Its attribute is the URL of the image we want to display on our page.
The URL points to the location where the image is stored, either on the computer or on the net.
the browser puts the image exactly where the image tag occurs.

The alt attribute is used to define an alternate text for the image. The value of the alt attribute is an author-defined text.
The alt attribute tells the reader what he or she is missing on a page if the browser cannot load the images.
When the cursor is kept for a few seconds on the image, then the browser will display the alternate text instead of the image.
